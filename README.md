# CSS Refresher

**Motivo:**  
Luego de refrescar mis conocimientos de HTML, me propongo **refrescar y reforzar mis conocimientos** de CSS, incluyendo layout, Flexbox y Grid.

**Objetivo:**  
Repasar CSS desde lo basico hasta avanzado, con ejemplos claros y progresivos, aplicando buenas practicas y técnicas modernas de diseño web.

**Contenido:**  
Este repositorio contiene **ejemplos propios organizados por niveles**. Cada carpeta representa un nivel de aprendizaje y contiene archivos HTML y CSS para practicar los conceptos correspondientes.

## Niveles

- **Nivel 1 — Fundamentos de CSS**  
  Sintaxis, selectores basicos, color, texto y comentarios.

- **Nivel 2 — Modelo de caja (Box Model)**  
  Margen, padding, borde, width, height, box-sizing.

- **Nivel 3 — Posicionamiento**  
  Static, relative, absolute, fixed, sticky y z-index.

- **Nivel 4 — Display y visibilidad**  
  Display (block, inline, inline-block, none), visibility.

- **Nivel 5 — Pseudoclases y pseudoelementos**  
  :hover, :focus, :active, :nth-child, ::before, ::after.

- **Nivel 6 — Transiciones y animaciones basicas**  
  Transition, transform, @keyframes.

- **Nivel 7 — Layout con Flexbox**  
  Contenedor flex, direccion, alineacion, flex items.

- **Nivel 8 — Layout con Grid**  
  Contenedor grid, filas, columnas, colocacion de items, gap.

- **Nivel 9 — Buenas practicas y CSS avanzado**  
  Variables, funciones, media queries, diseño responsivo y optimizacion.
